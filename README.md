# React SPA template

This is my personal React SPA setup, to quickly spin-up projects for experimentation.

## Usage

### pnpm

```bash
pnpm dlx degit cloud-walker/vite-react-spa my-project
```

### npm

```bash
npx degit cloud-walker/vite-react-spa my-project
```

### yarn 1

```bash
npx degit cloud-walker/vite-react-spa my-project
```
